# LeekScript v1

First version of LeekScript language, built in Java.
Used in the [leek-wars-generator-v1](https://github.com/leek-wars/leek-wars-generator-v1) project.

### Build
```
gradle jar
```
### Run a console
```
java -jar leekscript.jar
```
### Build and run tests
```
gradle jar test
```
